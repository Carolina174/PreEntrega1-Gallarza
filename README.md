# Mi e-comerce

```
Mi e-comerce surje como necesidad de un emprendimiento personal que tengo de personalización de elementos.En el mismo utilizo dos técnicas de estampado que se ven en el mercado, el estampado con vinilo y el estampado por transferencia de tinta por calor (sublimación.)
```

### Utilicé

- react
- firebase
- mui
- formik

```
1- React: es una biblioteca de JavaScript ampliamente utilizada para construir interfaces de usuario interactivas y reactivas. Permite dividir la interfaz de usuario en componentes reutilizables, lo que facilita la organización y el mantenimiento del código. React también utiliza la manipulación eficiente del DOM, lo que resulta en un mejor rendimiento y una experiencia de usuario más rápida.

2- Firebase: es una plataforma de desarrollo de aplicaciones móviles y web desarrollada por Google. Proporciona una amplia gama de herramientas y servicios, como autenticación de usuarios, almacenamiento en la nube, bases de datos en tiempo real, alojamiento web y más. Firebase permite crear aplicaciones en tiempo real y escalables sin la necesidad de administrar la infraestructura subyacente, lo que agiliza el desarrollo y permite centrarse en la funcionalidad principal de la aplicación.

3- Material-UI (MUI): es una biblioteca de componentes de interfaz de usuario diseñada para seguir las pautas de diseño de Material Design de Google. Proporciona componentes preestilizados y reutilizables que facilitan la creación de interfaces de usuario atractivas y coherentes. MUI ofrece una apariencia moderna y pulida, junto con funcionalidades como estilos personalizables, responsividad y temas, lo que ahorra tiempo y esfuerzo en el diseño y la implementación de la interfaz de usuario.

4- Formik: es una biblioteca diseñada para facilitar la gestión de formularios en React. Simplifica el proceso de captura y validación de datos del usuario, manejo de errores y presentación de mensajes de retroalimentación. Formik también ofrece integración con validación externa, como la biblioteca Yup, lo que permite crear formularios robustos y fiables con menos código y esfuerzo.
```
